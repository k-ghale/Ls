# Getting Started

## Prerequisites

- Git
* Node.js (v16 or higher)
+ npm or yarn

# Quick Setup

1. ### Clone the repository

```
git clone https://github.com/T4F25/backend.git
cd backend

```

2. ### Create a .env file

```
PORT=3000
DATABASE_URL=your_database_url
JWT_SECRET=your_secret_key
```

3. ### Install dependencies

```
npm install
```

4. ### Start the service

```
npm start
```

The API will be running on http://localhost:3000
